This is a little program for interacting with dogs.

To run it, just pull this repository and type in terminal `php index.php` from project directory.

All classes methods are covered by `PHPUnit` tests.
To run tests:
1. `composer install` for installing `PHPUnit` to the project.
2. `vendor/bin/phpunit tests/` to run all tests, or specify exact file from `test/` directory.
